This demo contains very basic examples of Angular functionality. The HTML and CSS in these examples has been kept as simple as possible, so you can focus on the Angular code.

Two main advantages of Angular:

  • It gets a lot done without much code
  • It doesn't depend on classes, IDs, parent-child relationships, or anything else that causes code to be brittle

This demo isn't a full survey of Angular's capabilities, just an idea of how it works.
