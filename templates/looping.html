Angular can loop through data. Consider the array of students and their grades below:

{{exampleArray}}

Angular can loop through that data. It's outside the scope of this demo, but Angular could also do something with an item in the array when the user clicks on its row in the table -- run a custom function that takes the item as a parameter, mark the item as the "selected" item, and so on.

This makes Angular powerful when it comes to tables and other structured sets of data.

We can put the data in a table ...

NAME GRADE
{{student.name}} {{student.grade}}

... or in an unordered list ...

  • {{student.name}}: {{student.grade}}

... or in any other repeating HTML structure ...

{{student.name}}

{{student.grade}}
